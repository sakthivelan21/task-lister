### Dependencies

❖ ejs  
❖ express  
❖ mysql2  
❖ nodemon  

**Install Dependencies**
 

```
# yarn
yarn install

# npm
npm install
```


**Starting Application**

```
npm start
```

+ Visit http://localhost:3000  to see the demo
